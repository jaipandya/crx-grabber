# CRX Grabber

## Project
Next.js app (App Router, TypeScript, Tailwind CSS) for downloading Chrome extension CRX files.

## Commands
- `bun dev` — start dev server
- `bun run build` — production build
- `bun run lint` — lint

## Conventions
- Use bun for everything (install, run, scripts)
- Keep it simple — single-page app, no backend DB, localStorage only
- All app code in `src/app/`
